# John Tech - Splash come Home

Questo pacchetto imposta **la splash page come HOME del sito**.

## Cosa fare in GitHub
1. Scarica ed estrai questo zip.
2. Nella tua repo `john.tech` rinomina il tuo attuale `index.html` in **main.html**.
3. Carica **index.html** (splash) e la cartella **assets/** di questo pacchetto.
4. Commit → attendi 1-2 minuti → visita `https://johntechroma.github.io/john.tech/`

Il pulsante "Entra nel sito" della splash punta a **main.html**.
